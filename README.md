📌 Project Overview

Financial fraud is one of the biggest challenges faced by banks and online payment systems. This project uses a Machine Learning model to classify credit card transactions as Legitimate or Fraudulent.

Users can upload a CSV dataset, and the application instantly:

1.Predicts fraudulent transactions

2.Displays fraud probability

3.Shows interactive visualizations

4.Generates transaction summaries

5.Allows downloading the prediction report


✨ Features

~ Credit Card Fraud Detection

~ Machine Learning Prediction

## ✨ Key Features

| 🚀 Feature | 📖 Description |
|------------|----------------|
| 📂 **CSV Upload** | Upload any transaction dataset instantly. |
| 🤖 **AI Prediction** | Logistic Regression classifies each transaction as legitimate or fraudulent. |
| 🚨 **Instant Alerts** | Displays warning alerts whenever fraudulent transactions are detected. |
| 📊 **Visual Analytics** | Interactive pie chart and bar chart for fraud analysis. |
| 📋 **Fraud Table** | Dedicated table displaying only flagged fraudulent transactions. |
| 📥 **Download Report** | Export the complete fraud analysis report as a CSV file. |
